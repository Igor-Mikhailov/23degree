Достаточно сложная верстка макета компании 3D-дизайнерской компании с диапазонами, ползунками, стилизованными tooltip-ами, выпадающим меню, уникальными полосами горизонтальной прокрутки и рисованием на адаптивном Canvas. Вполне развернутый как JQuery, так и Javascript-код.
Все полностью адаптивно, в процессе ограниченно использовался Bootstrap, JQuery, JQueryIU (компонент "ползунок диапазонов" в JQuery UI Slider), библиотека jScrollPane для создания адаптивного блока с горизонтальной прокруткой, библиотека popper.js для tooltip-ов, ну а также широко применялась SVG-графика для иконок путем вставки SVG-кода напрямую в HTML-разметку с дополнительными настройками параметров.
Основные бибилиотеки (и шрифты) подключены через CDN, дополнительные - находятся в папке "./libs" проекта.
При разработке и сборке файлов использовался возможно, устаревший, но надежный сборщик Gulp 4.
